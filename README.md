### 下载安装
-> [点击](https://github.com/bambooom/check4DUStat/releases/tag/v17.5.29)
- 下载 `chk4du.crx`
- chrome 中打开 `chrome://extensions/` 页面, 勾选右上角打开 `Developer mode`
- 将 `chk4du.crx` 拖入即可安装
